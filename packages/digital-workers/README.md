# digital-workers

Common abstract interface over AI Agents and Humans. Digital workers operate within a company/business boundary and share interfaces with Services, which can cross company boundaries.

## API

- `Role()` - Define a worker role
- `Team()` - Define a team of workers
- `Goals()` - Define worker goals
- `approve()` - Request approval
- `ask()` - Ask a question
- `do()` - Execute a task
- `decide()` - Make a decision
- `generate()` - Generate content
- `is()` - Type checking/validation
- `notify()` - Send notifications
- `kpis()` - Key performance indicators
- `okrs()` - Objectives and key results
