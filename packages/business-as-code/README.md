# business-as-code

Primitives for expressing business logic and processes as code.

## API

- `Business()` - Define a business entity
- `Vision()` - Define business vision
- `Goals()` - Define business goals
- `Product()` - Define products
- `Service()` - Define services
- `Process()` - Define business processes
- `Workflow()` - Define workflows
- `financials()` - Financial operations
- `kpis()` - Key performance indicators
- `okrs()` - Objectives and key results
- `$` - Financial/business operations helper
