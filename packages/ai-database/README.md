# ai-database

AI-powered database primitives for intelligent data storage and retrieval.

## API

- `DB()` - Database constructor
- `db.list()` - List records
- `db.find()` - Find records by criteria
- `db.search()` - Semantic search
- `db.get()` - Get a single record
- `db.set()` - Set a record value
- `db.create()` - Create a new record
- `db.update()` - Update an existing record
- `db.upsert()` - Create or update a record
- `db.delete()` - Delete a record
- `db.forEach()` - Iterate over records
