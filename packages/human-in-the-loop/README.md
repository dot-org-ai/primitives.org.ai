# human-in-the-loop

Primitives for integrating human oversight and intervention in AI workflows. Implements the digital-workers interface for humans operating within a company boundary.

## API

- `Human()` - Define human interaction points
- `Role()` - Define a human role
- `Team()` - Define a team of humans
- `Goals()` - Define human goals
- `approve()` - Request approval
- `ask()` - Ask a question
- `do()` - Request human action
- `decide()` - Request human decision
- `generate()` - Request content generation
- `is()` - Type checking/validation
- `notify()` - Send notifications
- `kpis()` - Key performance indicators
- `okrs()` - Objectives and key results
