# autonomous-agents

Primitives for building and orchestrating autonomous AI agents. Implements the digital-workers interface for AI agents operating within a company boundary.

## API

- `Agent()` - Create an autonomous agent
- `Role()` - Define an agent role
- `Team()` - Define a team of agents
- `Goals()` - Define agent goals
- `approve()` - Request approval
- `ask()` - Ask a question
- `do()` - Execute a task
- `decide()` - Make a decision
- `generate()` - Generate content
- `is()` - Type checking/validation
- `notify()` - Send notifications
- `kpis()` - Key performance indicators
- `okrs()` - Objectives and key results
