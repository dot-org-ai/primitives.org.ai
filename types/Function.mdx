> AI-powered capability that transforms input to output

## Type

```typescript
type Function = {
  name: string                        // Function name
  description?: string                // What it does
  input?: Record<string, unknown>     // Input schema
  output?: Record<string, unknown>    // Output schema
  examples?: Record<string, unknown>[] // Usage examples
  model?: string                      // AI model to use
  domain?: Domain                     // Parent namespace
  workflows?: Workflow[]              // Using workflows
  agents?: Agent[]                    // Using agents
}
```

| Property | Type | Description |
