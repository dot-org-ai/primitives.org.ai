> Autonomous AI actor that performs tasks

## Type

```typescript
type Agent = {
  name: string                                      // Agent name
  description?: string                              // What it does
  model?: string                                    // AI model used
  instructions?: string                             // System prompt
  status: 'idle' | 'working' | 'paused' | 'offline' // Current state
  capabilities?: string[]                           // What it can do
  domain?: Domain                                   // Parent namespace
  tools?: Tool[]                                    // Available tools
  functions?: Function[]                            // Available functions
  workflows?: Workflow[]                            // Assigned workflows
  tasks?: Task[]                                    // Assigned tasks
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Agent name |
| description | string? | What it does |
| model | string? | AI model used |
| instructions | string? | System prompt |
| status | idle \| working \| paused \| offline | Current state |
| capabilities | string[]? | What it can do |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| domain | Domain | Parent namespace |
| tools | Tool[] | Available tools |
| functions | Function[] | Available functions |
| workflows | Workflow[] | Assigned workflows |
| tasks | Task[] | Assigned tasks |

## Actions

| Action | Description |
|--------|-------------|
| create | Create an agent |
| update | Modify agent |
| deploy | Make available |
| start | Begin working |
| pause | Temporarily stop |
| stop | Shut down |
| assign | Give a task |

## Events

| Event | Description |
|-------|-------------|
| created | Agent was created |
| updated | Agent was modified |
| deployed | Agent went live |
| started | Agent began working |
| paused | Agent was paused |
| stopped | Agent shut down |
| assigned | Task was assigned |
