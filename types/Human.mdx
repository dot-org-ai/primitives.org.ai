> Person who interacts with the system

## Type

```typescript
type Human = {
  name: string                   // Full name
  email?: string                 // Email address
  role?: string                  // Job role
  status: 'active' | 'inactive'  // Account state
  permissions?: string[]         // Access rights
  projects?: Project[]           // Member projects
  tasks?: Task[]                 // Assigned tasks
  agents?: Agent[]               // Managed agents
  business?: Business            // Employer
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Full name |
| email | string? | Email address |
| role | string? | Job role |
| status | active \| inactive | Account state |
| permissions | string[]? | Access rights |
