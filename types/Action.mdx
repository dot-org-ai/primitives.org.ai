> Bound operation on a specific thing

## Type

```typescript
type Action = {
  name: string                                           // Action name
  status: 'pending' | 'running' | 'completed' | 'failed' // Current state
  input?: Record<string, unknown>                        // Input parameters
  output?: Record<string, unknown>                       // Result data
  startedAt?: Date                                       // Start timestamp
  completedAt?: Date                                     // Completion timestamp
  verb: Verb                                             // Operation type
  subject?: Thing                                        // Actor
  object: Thing                                          // Target
  event?: Event                                          // Resulting event
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Action name |
| status | pending \| running \| completed \| failed | Current state |
| input | json? | Input parameters |
| output | json? | Result data |
| startedAt | datetime? | Start timestamp |
| completedAt | datetime? | Completion timestamp |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| verb | Verb | Operation type |
| subject | Thing? | Actor |
| object | Thing | Target |
| event | Event? | Resulting event |

## Actions

| Action | Description |
|--------|-------------|
| create | Queue an action |
| execute | Run the action |
| cancel | Abort execution |
| retry | Re-execute failed action |

## Events

| Event | Description |
|-------|-------------|
| created | Action was queued |
| started | Execution began |
| completed | Execution succeeded |
| failed | Execution failed |
| cancelled | Action was aborted |
| retried | Action was retried |
