> Record of something that happened

## Type

```typescript
type Event = {
  type: string                       // Event type identifier
  timestamp: Date                    // When it occurred
  data?: Record<string, unknown>     // Event payload
  metadata?: Record<string, unknown> // Additional context
  domain?: Domain                    // Parent namespace
  verb?: Verb                        // Triggering operation
  subject?: Thing                    // Actor
  object?: Thing                     // Target
  action?: Action                    // Parent action
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| type | string | Event type identifier |
| timestamp | datetime | When it occurred |
| data | json? | Event payload |
| metadata | json? | Additional context |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| domain | Domain | Parent namespace |
| verb | Verb? | Triggering operation |
| subject | Thing? | Actor |
| object | Thing? | Target |
| action | Action? | Parent action |

## Actions

| Action | Description |
|--------|-------------|
| create | Emit an event |
| publish | Broadcast event |
| subscribe | Listen for events |
| replay | Re-emit past events |

## Events

| Event | Description |
|-------|-------------|
| created | Event was emitted |
| published | Event was broadcast |
| subscribed | Listener registered |
| replayed | Event was re-emitted |
