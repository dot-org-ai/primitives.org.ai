> Capability that agents and humans can use

## Type

```typescript
type Tool = {
  name: string                                         // Tool name
  description?: string                                 // What it does
  type: 'function' | 'api' | 'integration' | 'utility' // Tool category
  input?: Record<string, unknown>                      // Input schema
  output?: Record<string, unknown>                     // Output schema
  config?: Record<string, unknown>                     // Configuration
  domain?: Domain                                      // Parent namespace
  functions?: Function[]                               // Underlying functions
  agents?: Agent[]                                     // Using agents
}
```

## Properties

| Property | Type | Description |
