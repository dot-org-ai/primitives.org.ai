> Publication for content and updates

## Type

```typescript
type Blog = {
  name: string                   // Blog name
  description?: string           // What it's about
  status: 'active' | 'archived'  // Current state
  url?: string                   // Blog URL
  authors?: string[]             // Author names
  categories?: string[]          // Topic categories
  site?: Site                    // Parent site
  business?: Business            // Owner business
  posts?: Thing[]                // Blog posts
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Blog name |
| description | string? | What it's about |
| status | active \| archived | Current state |
| url | url? | Blog URL |
| authors | string[]? | Author names |
| categories | string[]? | Topic categories |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| site | Site | Parent site |
| business | Business | Owner business |
| posts | Thing[] | Blog posts |

## Actions

| Action | Description |
|--------|-------------|
| create | Create blog |
| update | Modify blog |
| publish | Post article |
| archive | Take offline |
| syndicate | Distribute content |

## Events

| Event | Description |
|-------|-------------|
| created | Blog was created |
| updated | Blog was modified |
| published | Article was posted |
| archived | Blog was taken offline |
| syndicated | Content was distributed |
