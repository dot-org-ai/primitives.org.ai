> Controlled test to validate a hypothesis

## Type

```typescript
type Experiment = {
  name: string                                            // Experiment name
  hypothesis: string                                      // What we're testing
  status: 'draft' | 'running' | 'completed' | 'cancelled' // Current state
  variants?: Record<string, unknown>[]                    // Test variations
  metrics?: string[]                                      // Success metrics
  startedAt?: Date                                        // Start timestamp
  endedAt?: Date                                          // End timestamp
  results?: Record<string, unknown>                       // Outcome data
  goal?: Goal                                             // Target objective
  project?: Project                                       // Parent project
  workflows?: Workflow[]                                  // Test workflows
}
```

| Property | Type | Description |
