> Strategy to achieve goals

## Type

```typescript
type Plan = {
  name: string                                           // Plan name
  description?: string                                   // Plan overview
  status: 'draft' | 'active' | 'completed' | 'cancelled' // Current state
  steps?: Record<string, unknown>[]                      // Planned steps
  startDate?: Date                                       // Start date
  endDate?: Date                                         // End date
  goals?: Goal[]                                         // Target outcomes
  projects?: Project[]                                   // Implementing projects
  tasks?: Task[]                                         // Planned tasks
  agents?: Agent[]                                       // Assigned agents
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Plan name |
| description | string? | Plan overview |
| status | draft \| active \| completed \| cancelled | Current state |
| steps | json[]? | Planned steps |
| startDate | date? | Start date |
| endDate | date? | End date |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|

## Actions

| Action | Description |
|--------|-------------|
| create | Create a plan |
| update | Modify the plan |
| execute | Start execution |
| revise | Update strategy |
| complete | Mark as done |
| cancel | Abandon plan |
