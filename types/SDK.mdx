> Developer kit for building integrations

## Type

```typescript
type SDK = {
  name: string                                                        // SDK name
  description?: string                                                // What it provides
  language: 'typescript' | 'python' | 'go' | 'rust' | 'java' | 'ruby' // Target language
  status: 'development' | 'beta' | 'stable' | 'deprecated'            // Current state
  version?: string                                                    // Current version
  repository?: string                                                 // Source repo
  api?: API                                                           // Parent API
  product?: Product                                                   // Parent product
  cli?: CLI                                                           // Related CLI
}
```
