> Action or operation that can be performed on things

## Type

```typescript
type Verb = {
  action: string                        // Infinitive form
  actor?: string                        // Role performing the action
  activity?: string                     // Gerund form (-ing)
  result?: string                       // Noun form of result
  inverse?: string                      // Opposite action
  preconditions?: Record<string, unknown> // Required conditions
  effect?: Record<string, unknown>      // State changes
  domain?: Domain                       // Parent namespace
  subject?: Noun                        // Actor type
  object?: Noun                         // Target type
  events?: Event[]                      // Emitted events
}
```
