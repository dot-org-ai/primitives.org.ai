> URL-based namespace that provides context for AI and data

## Type

```typescript
type Domain = {
  name: string         // Display name
  url: string          // Canonical URL
  description?: string // Human-readable description
  nouns?: Noun[]       // Type definitions
  verbs?: Verb[]       // Available operations
  things?: Thing[]     // Instances
  functions?: Function[] // AI capabilities
  agents?: Agent[]     // Autonomous actors
  workflows?: Workflow[] // Orchestrated processes
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Display name |
| url | url | Canonical URL |
| description | string? | Human-readable description |

## Relationships

| Relationship | Type | Description |

## Actions

| Action | Description |
|--------|-------------|
| create | Create a new domain |
| update | Update domain configuration |
| delete | Remove the domain |
| federate | Link with other domains |
| discover | Expose capabilities |
