> Curated collection of resources

## Type

```typescript
type Directory = {
  name: string                                         // Directory name
  description?: string                                 // What it contains
  type: 'resources' | 'tools' | 'people' | 'companies' // Content type
  status: 'draft' | 'live' | 'archived'                // Current state
  categories?: string[]                                // Category list
  filters?: Record<string, unknown>                    // Filter options
  site?: Site                                          // Parent site
  things?: Thing[]                                     // Listed items
}
```

| Property | Type | Description |
|----------|------|-------------|

| categories | string[]? | Category list |
