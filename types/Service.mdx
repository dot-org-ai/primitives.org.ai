> Capability delivered to customers

## Type

```typescript
type Service = {
  name: string                                          // Service name
  description?: string                                  // What it provides
  type: 'consulting' | 'managed' | 'support' | 'custom' // Service category
  status: 'available' | 'unavailable' | 'deprecated'    // Current state
  pricing?: Record<string, unknown>                     // Pricing model
  sla?: Record<string, unknown>                         // Service level agreement
  business?: Business                                   // Provider business
  products?: Product[]                                  // Related products
  customers?: Thing[]                                   // Customer instances
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Service name |
| description | string? | What it provides |
| type | consulting \| managed \| support \| custom | Service category |
| status | available \| unavailable \| deprecated | Current state |
| pricing | json? | Pricing model |
| sla | json? | Service level agreement |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| business | Business | Provider business |
| products | Product[] | Related products |
