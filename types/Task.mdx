> Unit of work to be done

## Type

```typescript
type Task = {
  name: string                                                      // Task name
  description?: string                                              // What to do
  status: 'todo' | 'in_progress' | 'blocked' | 'done' | 'cancelled' // Current state
  priority?: 'low' | 'medium' | 'high' | 'urgent'                   // Importance level
  dueDate?: Date                                                    // Deadline
  estimate?: number                                                 // Time estimate
  project?: Project                                                 // Parent project
  goal?: Goal                                                       // Contributing goal
  assignee?: Agent | Human                                          // Who's responsible
  dependencies?: Task[]                                             // Blocking tasks
  subtasks?: Task[]                                                 // Child tasks
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Task name |
| description | string? | What to do |
| status | todo \| in_progress \| blocked \| done \| cancelled | Current state |
| priority | low \| medium \| high \| urgent | Importance level |
| dueDate | date? | Deadline |
| estimate | number? | Time estimate |

## Actions

| Action | Description |
|--------|-------------|
| create | Create a task |
| assign | Assign to someone |
| start | Begin work |
| complete | Mark as done |
| block | Mark as blocked |
| cancel | Cancel task |

## Events

| Event | Description |
|-------|-------------|
| created | Task was created |
| assigned | Task was assigned |
| started | Work began |
| completed | Task was finished |
| blocked | Task was blocked |
| cancelled | Task was cancelled |
