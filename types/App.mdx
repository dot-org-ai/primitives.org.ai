> Application that users interact with

## Type

```typescript
type App = {
  name: string                                           // App name
  description?: string                                   // What it does
  type: 'web' | 'mobile' | 'desktop' | 'native'          // Platform type
  status: 'development' | 'beta' | 'live' | 'deprecated' // Current state
  url?: string                                           // App URL
  version?: string                                       // Current version
  product?: Product                                      // Parent product
  api?: API                                              // Backend API
  functions?: Function[]                                 // Used functions
  agents?: Agent[]                                       // Integrated agents
}
```

## Properties

| Property | Type | Description |
