> Organization that creates and delivers value

## Type

```typescript
type Business = {
  name: string                                                                   // Business name
  legalName?: string                                                             // Legal entity name
  type: 'startup' | 'smb' | 'enterprise' | 'agency' | 'nonprofit'                // Business type
  status: 'ideation' | 'launched' | 'growing' | 'scaling' | 'acquired' | 'closed' // Current state
  industry?: string                                                              // Industry sector
  website?: string                                                               // Company website
  domain?: Domain                                                                // Business namespace
  products?: Product[]                                                           // Product offerings
  services?: Service[]                                                           // Service offerings
  humans?: Human[]                                                               // Team members
  agents?: Agent[]                                                               // AI agents
  goals?: Goal[]                                                                 // Business goals
}
```
