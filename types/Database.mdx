> Persistent storage for things and relationships

## Type

```typescript
type Database = {
  name: string                                       // Database name
  type: 'graph' | 'relational' | 'document' | 'kv'  // Storage type
  url?: string                                       // Connection URL
  schema?: Record<string, unknown>                   // Schema definition
  domain?: Domain                                    // Parent namespace
  things?: Thing[]                                   // Stored instances
  nouns?: Noun[]                                     // Defined types
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Database name |
| type | graph \| relational \| document \| kv | Storage type |
| url | url? | Connection URL |
| schema | json? | Schema definition |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| domain | Domain | Parent namespace |
| things | Thing[] | Stored instances |
| nouns | Noun[] | Defined types |

## Actions

| Action | Description |
|--------|-------------|
| create | Create a database |
| connect | Establish connection |
| query | Execute queries |
| migrate | Update schema |
| backup | Create backup |
| restore | Restore from backup |

## Events

| Event | Description |
|-------|-------------|
| created | Database was created |
| connected | Connection established |
| queried | Query was executed |
| migrated | Schema was updated |
| backed_up | Backup was created |
| restored | Database was restored |
