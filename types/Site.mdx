> Web presence for content and commerce

## Type

```typescript
type Site = {
  name: string                                          // Site name
  description?: string                                  // What it's for
  type: 'marketing' | 'docs' | 'commerce' | 'community' // Site purpose
  status: 'development' | 'live' | 'archived'           // Current state
  url?: string                                          // Site URL
  analytics?: Record<string, unknown>                   // Tracking config
  product?: Product                                     // Parent product
  business?: Business                                   // Owner business
  blog?: Blog                                           // Attached blog
}
```
