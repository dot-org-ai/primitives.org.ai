> Interface for programmatic access

## Type

```typescript
type API = {
  name: string                                             // API name
  description?: string                                     // What it does
  type: 'rest' | 'graphql' | 'grpc' | 'websocket'          // Protocol type
  status: 'development' | 'beta' | 'stable' | 'deprecated' // Current state
  url?: string                                             // Base URL
  version?: string                                         // Current version
  spec?: Record<string, unknown>                           // OpenAPI/schema
  product?: Product                                        // Parent product
  functions?: Function[]                                   // Exposed functions
  sdk?: SDK                                                // Client SDK
}
```
