> Desired outcome to achieve

## Type

```typescript
type Goal = {
  name: string                                          // Goal name
  description?: string                                  // What to achieve
  status: 'draft' | 'active' | 'achieved' | 'abandoned' // Current state
  target?: Record<string, unknown>                      // Success criteria
  deadline?: Date                                       // Target date
  progress?: number                                     // Completion percentage
  plan?: Plan                                           // Parent strategy
  projects?: Project[]                                  // Related projects
  tasks?: Task[]                                        // Contributing tasks
  metrics?: Thing[]                                     // Tracking metrics
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Goal name |
| description | string? | What to achieve |
| status | draft \| active \| achieved \| abandoned | Current state |
| target | json? | Success criteria |
