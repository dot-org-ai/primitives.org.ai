> Organized effort to deliver an outcome

## Type

```typescript
type Project = {
  name: string                                                         // Project name
  description?: string                                                 // Project overview
  status: 'planning' | 'active' | 'paused' | 'completed' | 'cancelled' // Current state
  startDate?: Date                                                     // Start date
  endDate?: Date                                                       // End date
  budget?: number                                                      // Allocated budget
  goal?: Goal                                                          // Target outcome
  plan?: Plan                                                          // Parent plan
  tasks?: Task[]                                                       // Project tasks
  agents?: Agent[]                                                     // Assigned agents
  humans?: Human[]                                                     // Team members
}
```

## Properties

| Property | Type | Description |
|----------|------|-------------|
| name | string | Project name |
| description | string? | Project overview |
| status | planning \| active \| paused \| completed \| cancelled | Current state |
| startDate | date? | Start date |
| endDate | date? | End date |
| budget | number? | Allocated budget |

## Relationships

| Relationship | Type | Description |
|--------------|------|-------------|
| goal | Goal | Target outcome |
| plan | Plan | Parent plan |
| tasks | Task[] | Project tasks |
| agents | Agent[] | Assigned agents |
| humans | Human[] | Team members |

## Actions

| Action | Description |
|--------|-------------|
| create | Create a project |
| update | Modify project |
| start | Begin work |
| pause | Temporarily stop |
| complete | Mark as done |
| cancel | Abandon project |

## Events

| Event | Description |
|-------|-------------|
| created | Project was created |
| updated | Project was modified |
| started | Work began |
| paused | Work was paused |
| completed | Project was finished |
| cancelled | Project was abandoned |
