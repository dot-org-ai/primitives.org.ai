---
title: Equity
description: Types for cap tables, fundraising, equity compensation, and valuations
---

The `equity` module provides types for startup equity management.

```typescript
import type {
  Investor, Shareholder, ShareClass, Share,
  VestingSchedule, OptionGrant, FundingRound,
  CapTable, StartupMetrics, Valuation409A
} from '@primitives/types/equity'
```

## Types

| Type | Description |
|------|-------------|
| `Investor` | Investor entity (VC, angel, institutional) |
| `Shareholder` | Individual or entity holding shares |
| `ShareClass` | Class of shares (common, preferred series) |
| `Share` | Individual share certificate |
| `VestingSchedule` | Vesting terms (cliff, duration, acceleration) |
| `OptionGrant` | Stock option grant with exercise details |
| `FundingRound` | Fundraising round (seed, Series A, etc.) |
| `CapTable` | Full capitalization table |
| `StartupMetrics` | Key startup metrics (MRR, ARR, burn, runway) |
| `Valuation409A` | 409A valuation for tax purposes |
