Generate code in any language.

## Import

```typescript
import { code } from 'ai-functions'
```

## Syntax

```typescript
code`description`
code`description`(options)
code(description: string, options?: CodeOptions)
```

## Examples

```typescript
// Tagged template - concise
const validator = await code`email validation function`
const query = await code`SQL: top 10 customers by revenue`

// With complex requirements
const website = await code`marketing website${{
  requirements: {
    pages: ['home', 'about', 'pricing'],
    features: ['dark mode', 'animations'],
    stack: 'Next.js + Tailwind',
  },
}}`

// With explicit options
const component = await code('data table component', {
  language: 'tsx',
  framework: 'React',
  features: ['sorting', 'filtering', 'pagination'],
})
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `language` | string | Target language |
| `framework` | string | Framework/library |
| `features` | string[] | Required features |
| `style` | string | Code style preferences |

## Languages

```typescript
// Web
await code`React component for ${feature}`
await code`Vue composable for ${functionality}`
await code`CSS for ${design}`

// Backend
await code`Express route for ${endpoint}`
await code`Prisma schema for ${dataModel}`
await code`SQL migration for ${change}`

// Scripts
await code`bash script to ${task}`
await code`Python script for ${analysis}`
```

## With Context

```typescript
const brand = await ai.storyBrand({ hero: 'developers', guide: 'ai-functions' })
const landingPage = await code`marketing site for ${{ brand }}`
```
