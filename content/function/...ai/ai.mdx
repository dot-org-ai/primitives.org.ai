The simplest way to call AI. Pass a string, get a string back.

## Import

```typescript
import { ai } from 'ai-functions'
```

## Syntax

```typescript
ai`prompt with ${values}`
ai(prompt: string)
ai(prompt: string, options: AIOptions)
```

## Examples

```typescript
// Tagged template
const explanation = await ai`Explain ${topic} to a 5 year old`

// Direct call
const text = await ai('Explain quantum computing in one sentence')

// With options
const result = await ai`complex analysis of ${data}`({
  model: 'claude-opus-4-5',
  temperature: 0.7,
})
```

## Interpolation

Values in templates are converted automatically:

```typescript
// Primitives → string
ai`Write about ${topic}`  // "Write about TypeScript"

// Objects → YAML
ai`Analyze ${{ company, market }}`
// "Analyze\ncompany: Acme\nmarket: SaaS"

// Arrays → YAML list
ai`Compare ${['React', 'Vue', 'Svelte']}`
// "Compare\n- React\n- Vue\n- Svelte"
```

## Options

| Option | Type | Description |
|--------|------|-------------|
| `model` | `string` | Model to use |
| `temperature` | `number` | Randomness (0-1) |
| `maxTokens` | `number` | Max output tokens |

## When to Use

- Quick one-off generations
- Simple prompts without structure
- When you just need a string back
