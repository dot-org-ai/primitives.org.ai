Everything starts with `ai`. Call it, chain it, destructure it.

```typescript
import { ai } from 'ai-functions'

// Template literal
const explanation = await ai`Explain ${topic} to a 5 year old`

// Dynamic functions
const code = await ai.generateAuthMiddleware(requirements)
const analysis = await ai.analyzeMarketTrends(data)

// Destructuring
const { title, tags, excerpt } = ai`metadata for: ${blogPost}`
```

## Promise Pipelining

Chain operations without `await`â€”dependencies resolve automatically:

```typescript
const { score, qualified, reason } = ai`qualify ${lead}`

// Chain naturally
const email = ai`follow-up email based on: ${reason}`
const subject = ai`subject line for: ${email}`

// Only await when you need the value
if (await qualified) {
  await send({ subject: await subject, body: await email })
}
```

## Batch Processing

Process arrays in a single call:

```typescript
const leads = list`leads from ${campaign}`

const qualified = await leads.map(lead => ({
  lead,
  score: ai`score 1-100: ${lead}`,
  fit: ai`${lead} matches ${icp}?`,
}))
```

## All Functions

### Core

| Export | Description |
|--------|-------------|
| [`AI()`](/function/AI) | Create typed AI instances with predefined schemas |
| [`ai`](/function/ai) | The simplest way to call AI |
| [`define`](/function/define) | The foundation for creating AI functions |
| [`generate`](/function/generate) | The core generation primitive with schema support |

### Function Types

Every `ai.doSomething()` becomes one of four function types:

| Type | What It Does | Returns |
|------|--------------|---------|
| [Code](/function/code) | Generates executable code | Code string |
| [Generative](/function/generative) | Creates content/data | Structured data |
| [Agentic](/function/agentic) | Takes autonomous action | Action results |
| [Human](/function/human) | Requires human input | Human response |

### Generative

| Function | Description |
|----------|-------------|
| [`list`](/function/list) | Generate a list of items |
| [`lists`](/function/lists) | Generate multiple named lists |
| [`write`](/function/write) | Generate text content |
| [`summarize`](/function/summarize) | Summarize content |
| [`extract`](/function/extract) | Extract structured data |
| [`diagram`](/function/diagram) | Generate diagrams |
| [`slides`](/function/slides) | Generate presentations |
| [`image`](/function/image) | Generate images |
| [`video`](/function/video) | Generate video |

### Logic

| Function | Description |
|----------|-------------|
| [`is`](/function/is) | Boolean classification |
| [`decide`](/function/decide) | Make decisions from options |

### Action

| Function | Description |
|----------|-------------|

## Examples

### Sales Pipeline

```typescript
const leads = list`leads from ${campaign}`

const qualified = await leads.map(lead => ({
  lead,
  score: ai`score 1-100: ${lead}`,
  fit: ai`${lead} matches ${icp}?`,
}))

const approved = qualified.filter(l => l.score > 80)
const emails = await approved.map(l => write`follow-up for ${l.lead}`)
```

### Content Generation

```typescript
const ai = AI({
  blogPost: {
    title: 'Post title',
    summary: 'Brief summary',
    sections: [{
      heading: 'Section heading',
      content: 'Section content',
    }],
  },
})

const post = await ai.blogPost('TypeScript tips')
const summary = await summarize(post)
const tags = await list`tags for: ${post.title}`
```

### Code Generation

```typescript
const middleware = await ai.generateAuthMiddleware({
  provider: 'jwt',
  roles: ['admin', 'user', 'guest'],
})

const migration = await ai.createMigration({
  action: 'add users table',
  columns: ['id', 'email', 'name', 'created_at'],
})
```

### Human-in-the-Loop

```typescript
const proposal = await ai.generateProposal(requirements)
const approved = await approve`${proposal} for ${client}`

if (approved) {
  const feedback = await review`implementation plan for ${proposal}`
  const final = await ai.refineProposal(proposal, feedback)
}
```
