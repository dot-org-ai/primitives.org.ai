---
title: AI Functions
description: Call AI like a function. No prompts. No configuration.
---

Just call it.

```typescript
import { ai, is, list, code } from 'ai-functions'

await ai`write a haiku about ${topic}`
await is`${email} a valid email?`
await list`blog post ideas about ${topic}`
await code`${task} in ${language}`
```

## Installation

```bash
pnpm add ai-functions
```

## All Functions

### Core

| Function | Description |
|----------|-------------|
| [`ai`](/function/ai) | The simplest way to call AI |
| [`generate`](/function/generate) | Core generation primitive with schema support |
| [`define`](/function/define) | Foundation for creating AI functions |
| [`AI()`](/function/AI) | Create typed AI instances with schemas |

## Quick Examples

```typescript
// Generation - creates content, analyzes, extracts
const post = await write`blog post about ${topic}`
const ideas = await list`10 startup ideas for ${industry}`

// Classification - returns boolean or picks options
const isSpam = await is`${email} spam?`
const best = await decide`which is better?`(optionA, optionB)

// Code - generates executable code
const fn = await code`email validator in TypeScript`
const arch = await diagram`microservices architecture`

// Agentic - uses tools, multiple steps
const analysis = await research`${company} competitive landscape`

// Human - waits for human input
const decision = await ask`should we proceed with ${plan}?`
const ok = await approve`Deploy to production?`
```
