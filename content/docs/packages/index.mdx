# Packages

Primitives.org.ai is implemented as a collection of NPM packages. Each package provides specific functionality that can be used independently or composed together.

## Package Compatibility

All packages are designed to work together seamlessly:

```typescript
import { fn } from 'ai-functions'
import { workflow } from 'ai-workflows'
import { agent } from 'autonomous-agents'
import { service } from 'services-as-software'

// Functions compose into workflows
const myWorkflow = workflow({
  steps: [myFunction1, myFunction2],
})

// Agents use functions and workflows
const myAgent = agent({
  capabilities: [myFunction1, myWorkflow],
})

// Services wrap agents and workflows
const myService = service({
  implementation: { agent: myAgent },
})
```

## Installation

Install all core packages:

```bash

```

## TypeScript Support

All packages include TypeScript definitions and are designed for type-safe usage:

```typescript
import type { Function, Workflow, Agent, Service } from 'ai-functions'

// Full type inference
const myFn: Function<InputType, OutputType> = fn({
  name: 'typed-function',
  input: inputSchema,
  output: outputSchema,
})
```
