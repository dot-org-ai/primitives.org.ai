# Classification Functions

Make decisions and classifications with AI.

## Functions

| Function | Returns | Description |
|----------|---------|-------------|
| `is` | boolean | Yes/no classification |
| `decide` | T | Pick best option (LLM as judge) |

## Quick Examples

```typescript
import { is, decide } from 'ai-functions'

// is - boolean questions
const isSpam = await is`${email} spam?`
const isValid = await is`${code} valid TypeScript?`

// decide - pick the best
const best = await decide`which headline gets more clicks?`(
  'AI is Here',
  'The Future of AI',
  'Why AI Matters Now',
)
```

## When to Use

- **`is`** - Binary yes/no questions, content moderation, validation
- **`decide`** - A/B testing, content comparison, option selection
