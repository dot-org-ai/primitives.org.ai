# Agentic Functions

Functions that can use tools, browse the web, and perform multi-step tasks.

## Functions

| Function | Returns | Description |
|----------|---------|-------------|
| `do` | unknown | Single-pass task with tools |
| `research` | Research | Web research and synthesis |
| `read` | string | Fetch URL as markdown |
| `browse` | Page | Browser automation |

## Quick Examples

```typescript
import { do as act, research, read, browse } from 'ai-functions'

// do - task with tools
const analysis = await act`analyze article and extract key points ${article}`

// research - web research
const findings = await research`AI market trends 2025`

// read - URL to markdown
const content = await read`https://example.com/article`

// browse - browser automation
const page = await browse`https://competitor.com/pricing`
await page.do`click on Enterprise plan`
```
