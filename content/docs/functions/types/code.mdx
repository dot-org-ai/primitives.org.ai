# Code Functions

Generate code in any language, optionally execute in sandbox.

## Functions

| Function | Output | Example |
|----------|--------|---------|
| `code` | string | `code\`email validator in TypeScript\`` |
| `diagram` | string | `diagram\`authentication flow\`` |
| `slides` | string | `slides\`investor pitch\`` |
| `evaluate` | unknown | `evaluate({ code, tests })` |

## Characteristics

- **Language-aware** - Generates valid syntax
- **Executable** - Can run in sandbox (via ai-sandbox)
- **Structured** - Diagrams/slides in standard formats

## Examples

```typescript
import { code, diagram, slides } from 'ai-functions'

// Generate code
const validator = await code`email validation function`
const query = await code`SQL: top 10 customers by revenue`

// With requirements

// Generate diagrams

const erd = await diagram`e-commerce database schema`

// Generate presentations
const pitch = await slides`investor pitch for ${brand}`
```

## Sandbox Execution

```typescript
import { code } from 'ai-functions'
import { evaluate } from 'ai-sandbox'

const fn = await code`fibonacci function in TypeScript`
const result = await evaluate({
  code: fn,
  tests: [
    { input: 10, expected: 55 },
    { input: 20, expected: 6765 },
  ],
})
```
