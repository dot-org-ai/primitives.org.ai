# Noun Properties

Define properties with types, descriptions, and constraints.

## Property Types

| Type | Description |
|------|-------------|
| `'string'` | Text |
| `'number'` | Numeric |
| `'boolean'` | True/false |
| `'date'` | Date only |
| `'datetime'` | Date and time |
| `'json'` | JSON object |
| `'markdown'` | Markdown text |
| `'url'` | URL string |

## Basic Properties

```typescript
properties: {
  title: { type: 'string' },
  count: { type: 'number' },
  active: { type: 'boolean' },
  publishedAt: { type: 'datetime' },
  content: { type: 'markdown' },
  website: { type: 'url' },
  settings: { type: 'json' },
}
```

## With Description

Descriptions guide AI generation:

```typescript
properties: {
  title: {
    type: 'string',
    description: 'Compelling blog post title (50-70 characters)',
  },
  excerpt: {
    type: 'string',
    description: 'Hook the reader in 2-3 sentences',
  },
  content: {
    type: 'markdown',
    description: 'Full article with headers, code examples, and insights (1500-2500 words)',
  },
}
```

## Optional Properties

```typescript
properties: {
  title: { type: 'string' },                    // required
  subtitle: { type: 'string', optional: true }, // optional
}
```

## Array Properties

```typescript
properties: {
  tags: {
    type: 'string',
    array: true,
    description: '3-5 relevant tags',
  },
  scores: {
    type: 'number',
    array: true,
  },
}
```

## Default Values

```typescript
properties: {
  status: {
    type: 'string',
    default: 'draft',
  },
  views: {
    type: 'number',
    default: 0,
  },
  featured: {
    type: 'boolean',
    default: false,
  },
}
```

## Examples

Provide examples for documentation and generation:

```typescript
properties: {
  tags: {
    type: 'string',
    array: true,
    examples: ['typescript', 'react', 'tutorial'],
  },
  rating: {
    type: 'number',
    examples: [4.5, 3.8, 5.0],
  },
}
```

## Enum-Like Properties

```typescript
properties: {
  status: {
    type: 'string',
    description: 'draft | published | archived',
  },
  priority: {
    type: 'string',
    description: 'low | medium | high | critical',
    default: 'medium',
  },
}
```

## Nested JSON

```typescript
properties: {
  metadata: {
    type: 'json',
    description: 'Additional metadata object',
  },
  settings: {
    type: 'json',
    default: { theme: 'light', notifications: true },
  },
}
```

## Real-World Examples

### Blog Post

```typescript
properties: {
  title: {
    type: 'string',
    description: 'Engaging title that promises value',
  },
  slug: {
    type: 'string',
    description: 'URL-safe identifier (auto-generated from title)',
  },
  excerpt: {
    type: 'string',
    description: 'Compelling preview (150-200 chars)',
    optional: true,
  },
  content: {
    type: 'markdown',
    description: 'Full article body',
  },
  status: {
    type: 'string',
    description: 'draft | published | archived',
    default: 'draft',
  },
  publishedAt: {
    type: 'datetime',
    optional: true,
  },
  tags: {
    type: 'string',
    array: true,
    examples: ['typescript', 'tutorial', 'best-practices'],
  },
  readTime: {
    type: 'number',
    description: 'Estimated read time in minutes',
    optional: true,
  },
}
```
