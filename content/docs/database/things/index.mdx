# Things

Everything in ai-database is a **Thing**. A Thing is an entity with a typed schema, unique ID, and automatic relationships.

```typescript
const db = DB({
  Post: { title: 'string', content: 'markdown' },
  Author: { name: 'string', email: 'string' }
})

// Every entity is a Thing
const post = await db.Post.create({ title: 'Hello' })
post.$id    // 'abc123'
post.$type  // 'Post'
post.title  // 'Hello'
```

## Entity Structure

Every Thing has these built-in fields:

| Field | Type | Description |
|-------|------|-------------|
| `$id` | `string` | Unique identifier |
| `$type` | `string` | Type name |
| `createdAt` | `Date` | Creation timestamp |
| `updatedAt` | `Date` | Last update timestamp |

Plus your schema-defined fields.

## Type Safety

Schemas provide full TypeScript inference:

```typescript
const db = DB({
  User: {
    name: 'string',
    email: 'string',
    age: 'number?',
  }
})

// Fully typed
const user = await db.User.create({
  name: 'Alice',
  email: 'alice@example.com',
  // age is optional
})

user.name   // string
user.email  // string
user.age    // number | undefined
```

## Self-Describing
