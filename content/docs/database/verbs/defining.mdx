# Defining Verbs

Create rich action definitions with `defineVerb`.

## Basic Definition

```typescript
import { defineVerb } from 'ai-database'

const publish = defineVerb({
  action: 'publish',
})
```

## With All Forms

```typescript
const publish = defineVerb({
  action: 'publish',
  actor: 'publisher',
  act: 'publishes',
  activity: 'publishing',
  result: 'publication',
  reverse: {
    at: 'publishedAt',
    by: 'publishedBy',
    in: 'publishedIn',
    for: 'publishedFor',
  },
  inverse: 'unpublish',
  description: 'Make content publicly available',
})
```

## Verb Interface

```typescript
interface Verb {
  /** Base form / imperative (create, publish) */
  action: string

  /** Agent noun (creator, publisher) */
  actor?: string

  /** Present tense 3rd person (creates, publishes) */
  act?: string

  /** Present participle / gerund (creating, publishing) */
  activity?: string

  /** Result noun (creation, publication) */
  result?: string

  /** Reverse/passive forms */
  reverse?: {
    at?: string    // createdAt, publishedAt
    by?: string    // createdBy, publishedBy
    in?: string    // createdIn, publishedIn
    for?: string   // createdFor, publishedFor
    [key: string]: string | undefined
  }

  /** Inverse action (delete, unpublish) */
  inverse?: string

  /** Description of action */
  description?: string
}
```

## Using Verbs

### In Nouns

```typescript
const Post = defineNoun({
  singular: 'post',
  plural: 'posts',
  actions: [
    'create',
    'update',
    'delete',
    publish,  // custom verb
  ],
})
```

### For Field Names

```typescript
const Verbs = { create, update, delete: deleteVerb, publish }

// Get reverse fields
Verbs.create.reverse.at   // 'createdAt'
Verbs.create.reverse.by   // 'createdBy'
Verbs.publish.reverse.at  // 'publishedAt'
```

### For Event Names

```typescript
const eventType = `Post.${publish.action}d`  // 'Post.published'
```

## Custom Verbs

Define domain-specific verbs:

```typescript
const approve = defineVerb({
  action: 'approve',
  actor: 'approver',
  act: 'approves',
  activity: 'approving',
  result: 'approval',
  reverse: { at: 'approvedAt', by: 'approvedBy' },
  inverse: 'reject',
})

const submit = defineVerb({
  action: 'submit',
  actor: 'submitter',
  act: 'submits',
  activity: 'submitting',
  result: 'submission',
  reverse: { at: 'submittedAt', by: 'submittedBy' },
})

const review = defineVerb({
  action: 'review',
  actor: 'reviewer',
  act: 'reviews',
  activity: 'reviewing',
  result: 'review',
  reverse: { at: 'reviewedAt', by: 'reviewedBy' },
})
```


## Action Flow

Model a workflow:

```typescript
const draft = defineVerb({ action: 'draft' })
const submit = defineVerb({ action: 'submit', inverse: 'withdraw' })
const review = defineVerb({ action: 'review' })
const approve = defineVerb({ action: 'approve', inverse: 'reject' })
const publish = defineVerb({ action: 'publish', inverse: 'unpublish' })

// Status flow: draft → submitted → reviewed → approved → published
```

## Helper: getVerbFields

Get reverse fields for a standard verb:

```typescript
import { getVerbFields } from 'ai-database'

getVerbFields('create')
// { at: 'createdAt', by: 'createdBy', in: 'createdIn', for: 'createdFor' }

getVerbFields('publish')
// { at: 'publishedAt', by: 'publishedBy' }
```
