# Queries

ai-database unifies three query paradigms in a single API. Use whichever style fits your data modelâ€”or combine all three.

## Query Styles

| Style | Best For | Example |
|-------|----------|---------|
| [SQL](/docs/database/queries/sql) | Relational data, comparisons | `price: { $gt: 100 }` |
| [Document](/docs/database/queries/document) | Nested data, arrays | `'profile.city': 'SF'` |
| [Graph](/docs/database/queries/graph) | Relationships, traversal | `$rel: { type: 'follows' }` |

## Basic Queries

```typescript
// List all
const posts = await db.Post.list()

// Find by criteria
const published = await db.Post.find({
  status: 'published',
})

// With options
const recent = await db.Post.list({
  where: { published: true },
  orderBy: 'createdAt',
  order: 'desc',
  limit: 10,
})
```

## Combining Styles

Mix all three in a single query:

```typescript
const posts = await db.Post.find({
  // SQL-style: comparisons
  publishedAt: { $gte: lastWeek },

  // Document-style: nested fields
  'metadata.wordCount': { $gte: 1000 },
  tags: { $containsAll: ['ai', 'tutorial'] },

  // Graph-style: relationships
  $rel: {
    type: 'authored',
    from: {
      type: 'User',
      where: { verified: true },
    },
  },
})
```

## Operators

See [Operators](/docs/database/queries/operators) for the complete reference:

| Category | Operators |
|----------|-----------|
| Comparison | `$eq`, `$ne`, `$gt`, `$gte`, `$lt`, `$lte` |
| Set | `$in`, `$nin` |
| Array | `$contains`, `$containsAll`, `$size` |
| String | `$startsWith`, `$endsWith`, `$regex` |
| Existence | `$exists`, `$type` |
| Logical | `$and`, `$or`, `$not` |
| Graph | `$rel`, `$relCount` |

## Search vs Query

**Queries** filter by exact criteria:

```typescript
const drafts = await db.Post.find({
  status: 'draft',
})
```

**Search** finds by semantic similarity:

```typescript
const results = await db.Post.search('AI tutorials for beginners')
```

Combine both:

```typescript
const results = await db.Post.search({
  query: 'AI tutorials',
  where: { published: true },
})
```
