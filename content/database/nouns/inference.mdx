# Noun Inference

ai-database auto-generates linguistic forms for nouns.

## inferNoun

Infer a complete Noun from a type name:

```typescript
import { inferNoun } from 'ai-database'

inferNoun('Post')
// {
//   singular: 'post',
//   plural: 'posts',
//   actions: ['create', 'update', 'delete'],
//   events: ['created', 'updated', 'deleted'],
// }

inferNoun('BlogPost')
// {
//   singular: 'blog post',
//   plural: 'blog posts',
//   ...
// }

inferNoun('Category')
// {
//   singular: 'category',
//   plural: 'categories',
//   ...
// }
```

## pluralize

Auto-pluralize any noun:

```typescript
import { pluralize } from 'ai-database'

// Regular
pluralize('post')      // 'posts'
pluralize('box')       // 'boxes'
pluralize('buzz')      // 'buzzes'

// -y endings
pluralize('category')  // 'categories'
pluralize('city')      // 'cities'
pluralize('day')       // 'days' (vowel + y)

// -f/-fe endings
pluralize('leaf')      // 'leaves'
pluralize('knife')     // 'knives'
pluralize('half')      // 'halves'

// Irregular
pluralize('person')    // 'people'
pluralize('child')     // 'children'
pluralize('mouse')     // 'mice'
pluralize('datum')     // 'data'
pluralize('criterion') // 'criteria'
```

## singularize

Reverse pluralization:

```typescript
import { singularize } from 'ai-database'

singularize('posts')      // 'post'
singularize('categories') // 'category'
singularize('leaves')     // 'leaf'
singularize('people')     // 'person'
singularize('children')   // 'child'
singularize('data')       // 'datum'
```

## Type

Get full type metadata:

```typescript
import { Type } from 'ai-database'

const Post = Type('Post')
Post.name       // 'Post'
Post.singular   // 'post'
Post.plural     // 'posts'
Post.slug       // 'post'
Post.slugPlural // 'posts'

// Event types
Post.created    // 'Post.created'
Post.updated    // 'Post.updated'
Post.deleted    // 'Post.deleted'

// Verb-derived fields
Post.creator    // 'creator'
Post.createdAt  // 'createdAt'
Post.createdBy  // 'createdBy'
Post.updatedAt  // 'updatedAt'
Post.updatedBy  // 'updatedBy'
```

### Compound Names

```typescript
const BlogPost = Type('BlogPost')
BlogPost.singular   // 'blog post'
BlogPost.plural     // 'blog posts'
BlogPost.slug       // 'blog-post'
BlogPost.slugPlural // 'blog-posts'

const UserProfile = Type('UserProfile')
UserProfile.singular // 'user profile'
UserProfile.slug     // 'user-profile'
```

## TypeMeta Interface

```typescript
interface TypeMeta {
  name: string
  singular: string
  plural: string
  slug: string
  slugPlural: string
  creator: string
  createdAt: string
  createdBy: string
  updatedAt: string
  updatedBy: string
  created: string
  updated: string
  deleted: string
}
```

## Usage in Events

```typescript
db.events.on(Type('Post').created, async (event) => {
  console.log(`New ${Type('Post').singular}: ${event.data.title}`)
})
```

## Usage in Routes

```typescript
// Generate routes from type metadata
const type = Type('BlogPost')

app.get(`/${type.slugPlural}`, listPosts)        // /blog-posts
app.get(`/${type.slugPlural}/:id`, getPost)      // /blog-posts/:id
app.post(`/${type.slugPlural}`, createPost)      // /blog-posts
```

## Caching

Type metadata is cached:

```typescript
const a = Type('Post')
const b = Type('Post')
a === b  // true (same cached instance)
```

## AI-Powered Inference

For unknown words, ai-database uses rules-based inference. In the future, AI will be used to infer forms and cache them:

```typescript
// Coming soon:

```

Current rules handle:
- Regular plurals (add -s, -es)
- -y → -ies
- -f/-fe → -ves
- Common irregulars (person, child, etc.)
